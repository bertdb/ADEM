= Arduino setup

The firmware is written in Arduino.


== Installing the ESP8266 Arduino Addon

Follow the SparkFun tutorial on how to install the ESP8266 to your Arduino IDE:
https://learn.sparkfun.com/tutorials/esp8266-thing-hookup-guide/installing-the-esp8266-arduino-addon


== Included Arduino libraries

All external libraries we depend upon for the firmware are included in the repository.
This to ensure that we have reproducible builds.

The following libraries are being used:

  - License: LGPL v3
  - URL: https://github.com/adafruit/Adafruit_NeoPixel

ArduinoJson::

espsoftwareserial::

TinyGPS++::
